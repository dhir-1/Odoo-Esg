# EcoSphere Frontend

EcoSphere is a React + Vite + TypeScript dashboard for tracking environmental, social, governance, and gamified sustainability performance.

## Tech stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- react-router-dom

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
